# bench

This program benchmarks some languages with quick sort algorithm.

## Requirement
- bash (+ timeout command)
- C++ 11 or later
- Java 8 or later
- Python 2
- Go 1.6 or later

## Usage
`$ ./bench.sh ${TIMEOUT} ${TRIALS}`
- ${TIMEOUT}: timeout of each benchmark program
- ${TRIALS}: count of trials for each job
